# Sadguru Developers - Real Estate Website

A modern, responsive real estate website with a public-facing customer portal and an admin panel for property management.

## Features

### Public Website
- Modern, animated UI with glassmorphism effects
- Responsive design (mobile-first approach)
- Property listing with hover animations
- Property filtering by location, price, and type
- Contact form with validation
- Testimonial section
- Interactive navigation

### Admin Panel
- Secure login system
- Dashboard with statistics
- Add, edit, delete properties
- View and manage customer inquiries
- Responsive admin interface
